Picker_Date.js简介

Picker_Date.js日期选择格式为YYYY-MM-dd的Picker组件， 绑定在yyq_components原型上的picker_date对象.
说明:使用前请先调用initData(option)初始化数据，必须参数 option表示组件的配置项，包含有autoHide字段，默认值是true表示点击组件以外的部分是否点击组件外部自动隐藏组件。以及ConfirmCallback字段，表示点击确定之后的回调函数，类型Function,返回选中的值。以及CancelCallback字段，表示点击取消之后调用的回调函数，类型Function 以及MaxYear表示最大的年份，默认值为当前年份加上5，以及参数MinYear,表示最小年份，默认值是当前年份减去50.
初始化完成之后，每次显示都只需要调用show(val),val传入初始化值，可以是一个Date类型，也可以是一个Date String类型
//示例:
//调用显示前，先初始化数据
yyq_components.picker_date.initData({
MaxYear:2020,
MinYear:1993,
ConfirmCallback:function(val){
if(new Date(val.replace(/-/g,"/"))<=new Date()){
console.error("不能小于当前日期哦");
return false;
}
}
});
//显示组件，如果组件的原始数据没有变，那么只需要初始化一次数据就可以
yyq_components.picker_date.show("1993-02-05");
