Popup_Input.js简介

Popup_Input.js是一个弹窗输入框组件， 绑定在yyq_components原型的popup_input对象。
说明：show(option),用于显示组件，必须参数option是一个Object,里面的key有： Title用来显示标题；Content用来显示输入框的值，也就是初始值；InputType表示输入框的类型，默认值是text类型；ConfirmContent,用来显示弹窗右下侧按钮的文字，默认值是"确定"；CancelContent用来显示左下侧按钮的文字，默认值是"取消"；ConfirmCallback表示点击右下侧按钮的回调函数，返回输入框的值，点击该按钮默认会关闭弹窗，如果该回调函数执行完毕返回false，将不会关闭弹窗；CancelCallback表示点击左下侧按钮的回调函数，原理同ConfirmCallback一样
yyq_components.popup_input.show({
Title:"姓名",
Content:"崔海峰",
CancelContent:"清除",
ConfirmCallback:function(val){
//do something....
}
});
